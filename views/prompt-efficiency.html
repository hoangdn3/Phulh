Prompt Efficiency Tips

Writing a good prompt is not just about asking questions — It's about structuring your thinking.

  • Save time by getting better outputs in fewer iterations
  • Learn how to prompt like a consultant / strategist
  • Build a reusable mental model for all prompt-based tasks

The Prompt Pyramid: 3 Building Blocks

[ ROLE ]
You are a senior HR recruiter at a tech company.
[ CONTEXT ]
I am a GMBA student applying for a Product Manager job.
[ TASK ]
Please write a 3-sentence summary for my Linkein headline.

Tips to Maximize Prompt Efficiency

Tip 1: Always assign a role

+

Give the AI a clear professional identity — this helps set tone, format, and expected domain.

Example Prompt:
"You are a senior recruiter at Google. Review my resume and generate 3 bullet points that match the job description."

Tip 2: Define expected output

+

Don't just ask for help — say what format you need (bullets? paragraph?), tone, and any limits (e.g., word count).

Example Prompt:
"Summarize this article in 5 bullet points, using a confident MBA tone, and under 80 words."

Tip 3: Add just enough context

+

AI doesn't need your life story — just who you are, your purpose, and audience/task.

Example Prompt:
"I'm a GMBA student preparing a 3-minute pitch on digital banking adoption in Vietnam for a Strategy class."

Tip 4: Use the 3-layer structure

+

Structure all prompts like this:
[Role] + [Context] + [Task] → Helps AI think in frames.

Example Prompt:
"You're a McKinsey consultant. I'm researching market entry options into Thailand for an EdTech product. Suggest 3 frameworks and recommend one."

Tip 5: Break complex tasks into steps

+

For unclear or multi-layered tasks, ask step by step. You'll save time and control quality better.

Example Prompt:
"First, create an outline for a LinkedIn post about my summer internship at KPMG. Then expand the first point into 3 persuasive sentences."

Now try using this structure with your own task; writing an email / summarizing a case / simulating an interview.
